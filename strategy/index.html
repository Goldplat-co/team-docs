1. Gleami 린 캔버스 v1.0

우리 사업의 핵심 가정과 구조를 한 장에. ❓ 표시 = 검증 필요한 가정

2 Problem

  • P1. 선택의 한계 — 금은방은 그 가게 것만 볼 수 있음. 종로 2,000개 업체인데 2~3곳만 방문 가능
  • P2. 가격 불투명 — "이 가격이 적정한가?" 확인 방법 없음. 금시세 대비 공임이 얼마인지 모름
  • P3. 온라인 부재 — 골드 주얼리 전문 온라인 채널 자체가 존재하지 않음. 실버/패션과 뒤죽박죽
기존 대안
  • 금은방 (오프라인, 비교불가)
  • 홈쇼핑 (시간제한, 선택제한)
  • 네이버쇼핑 (뒤죽박죽, 신뢰↓)

4 Solution

  • S1. 큐레이션 — 종로 여러 소싱처의 골드 주얼리를 한 곳에 모아 셀렉트 (60 → 2,000개)
  • S2. 콘텐츠 — 보세를 프리미엄으로 보여주는 고퀄 촬영·착용샷·상세페이지
  • S3. 가격 경쟁력 — 제조사 D2C 직거래, 중간마진 제거. 금은방보다 합리적 가격

🔴 #1 치명적
"3040 환금성 고객이 온라인에서 보세 골드 주얼리를 산다"
검증: 런칭 후 실제 구매자 프로필 확인. 구매자 5명 전화 인터뷰.
🟠 #2 높음
"골드 주얼리 전문 셀렉트샵 포지셔닝에 고객이 반응한다"
검증: 광고 메시지 A/B 테스트. CTR + 랜딩 체류시간 비교.
🟠 #3 높음
"이 타겟(3040)을 디지털 채널로 획득할 수 있다"
검증: 채널별(인스타/네이버/카카오) CPA 비교. 퀸잇은 TV+카톡으로 4050을 잡았음.
핵심 인사이트
종로 플랫폼이 지금까지 없었던 이유는 수요가 없어서가 아니라, 공급을 풀 수 없어서다.
범준이사님(종로 15년+ 왕도매)이 있는 Gleami는 이 공급 장벽을 유일하게 넘은 팀. 나머지는 수요 검증의 문제.

4. 마케팅 전략 — 유저 여정별 설계

1단계에서 욕망을 점화하고, 2단계에서 불안을 해소하는 2단 구조

Stage 1
발견 — "어? 이쁘다"
비주얼 후킹으로 감정 점화
"갖고 싶다" "나한테 어울리겠다"

채널: 인스타, 숏폼, 네이버
콘텐츠: 착용샷, 스타일링, 무드
Stage 2
탐색 — "여기 뭐지?"
사이트 진입 후 둘러봄
"다른 것도 이쁘네" "가격이?"

핵심: 상품 다양성, 카테고리
목표: 3개+ 상품 탐색 유도
Stage 3
검토 — "사도 되나?"
불안 해소 + 확신 부여
"종로 직소싱이라" "진짜 금이고"

핵심: 신뢰 시그널
도구: 후기, 스펙, 보증, 반품정책
Stage 4
구매 — "질렀다!"
결제 마찰 최소화
간편결제, 할부, 무료배송

후속: 언박싱 경험
목표: 재방문 유도

🎯 감정 트리거 테스트 Phase 1 — 런칭 전/직후

같은 제품 사진, 메시지만 다르게 → 어떤 감정이 먹히는지 확인
  • T1. 착용 욕구 — 모델컷 중심. "이게 나한테 올라간 모습" 상상 유도
  • T3. 발견의 쾌감 — "종로에서 건져온 디자인" "아직 아무도 모르는"
  • T4. 가격 놀라움 — "이 퀄리티에 이 가격?" 비주얼+가격 태그만
  • 메타(인스타) 광고 3종, 5~10만원 예산, 3~5일 → CTR로 승자 판별

📱 채널 전략 Phase 1

3040이 실제로 있는 곳에서 잡아야 한다
  • 인스타그램 — 비주얼 콘텐츠 메인. 릴스/피드 동시 운영
  • 네이버 블로그/SEO — "14K 목걸이 추천" "골드 반지 인기" 등 검색 타겟
  • 카카오 모먼트 — 3040 여성 타겟 정밀 광고
  • 초기 예산 배분: 인스타 50% / 네이버 30% / 카카오 20% → 데이터 보고 조정

📸 콘텐츠 전략 Phase 1

Gleami의 핵심 무기. 보세를 보세답지 않게 보여주는 것
  • 제품 촬영 — 착용샷(필수), 디테일컷, 360도, 사이즈 비교
  • 스타일링 콘텐츠 — "이 목걸이 + 이 코디" 조합 제안
  • 숏폼 영상 — 착용 순간, 빛 반사, 언박싱. 15~30초
  • 촬영 프로세스: 렌탈 → 촬영(1~2일) → 반납 → 편집 → 업로드

🤝 신뢰 구축 전략 Phase 2 — 런칭 후 1~2개월

경쟁 비교에서 유일한 약점 = "신뢰". 이걸 빠르게 쌓아야 함
  • "종로 직소싱" 스토리 — 종로 촬영 현장, 소싱 과정을 콘텐츠로
  • 100% 교환/반품 보장 — "마음에 안 들면 무조건 반품" 선언
  • 실구매자 후기 — 구매 인증샷 + 착용 후기 적극 수집
  • 금 함량 인증 — 제품별 함량/중량 표기. Phase 2에서 인증서 도입 검토

📞 정성 검증 (직접 물어보기)

구매자 5명 전화 인터뷰 — 이게 가장 중요
  • Q1. "처음에 뭐 보고 들어오셨어요?" → 트리거 확인
  • Q2. "장바구니 담고 바로 결제? 고민?" → 불안 수준 확인
  • Q3. "뭐 때문에 결국 사셨어요?" → 핵심 구매 이유 = One Thing의 실마리
  • Q4. "Gleami 없어지면 좀 아쉬울 것 같으세요?" → Sean Ellis 간이 테스트
오늘의 결론
Gleami의 "One Thing"은 머리에서 나오는 게 아니라, 고객 입에서 나온다.
런칭 후 Q3("뭐 때문에 결국 사셨어요?")의 답에서 One Thing의 실마리를 찾자.
무신사의 랭킹도, 퀸잇의 4050 브랜드도 사전 설계가 아닌 고객 반응에서 발견된 것.
지금 할 수 있는 최선 = 가설을 빠르게 시장에 던지고, 고객이 뭐라 하는지 듣는 것.
