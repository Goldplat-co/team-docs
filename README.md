# GoldPlat Team Docs

> 팀원 및 외부 공유용 공개 문서 레포. GitHub Pages로 URL 공유 가능.

## 용도
- 팀 미팅 자료, 전략 캔버스 등 **URL로 공유가 필요한 문서**를 관리
- 민감 정보(API 키, 내부 전략 상세 등)는 포함하지 않음
- 원본은 `company-context`(private)에 보관, 공유용 버전만 여기에 게시
